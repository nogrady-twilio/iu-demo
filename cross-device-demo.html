This demo shows how Segment can recognize the same user across different devices using a hashed email as the user_id.

How It Works:

  • Consistent User ID: Each email generates the same user_id hash every time
  • Cross-Device Tracking: Same email on laptop, phone, tablet = same user_id
  • Privacy-Friendly: Email is hashed, not stored in plain text
  • Segment Integration: All events are tied to the same user profile
