üéØ Segment Implementation Details

How Both Emails Become Identifiers:

// When student signs up
const userId = hashEmail(personalEmail);  // Generate consistent ID

// Send to Segment with both emails
analytics.identify(userId, {
    email: personalEmail,           // Primary identifier
    email_personal: personalEmail,  // Custom trait
    email_iu: iuEmail,             // University email
    // ... other traits
});

// Create aliases for both emails
analytics.alias(personalEmail);    // Links personal email ‚Üí userId
analytics.alias(iuEmail);         // Links IU email ‚Üí userId
